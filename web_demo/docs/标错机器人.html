读取数据表：
image_match_result
读取的条件：
robot_processed = 0
处理逻辑：
把image_match_result中的图片匹配对取出来，计算两个图片的sift相似度，然后设置下面几个属性：
robot_score (扩展位 double)  = 相似度
robot_match (扩展位 int)  = 是否匹配
robot_processed （扩展位 int） = 1
同时需要改造：图片对转商品对的逻辑，从之前的“以image_match_result的added_to_item_match为标志位，处理过的1，未处理过的0”，变成“以image_match_result的added_to_item_match 和robot_processed 、robot_match为标志位，处理所有robot_processed = 1 && added_to_item_match = 0 && robot_match = 1”的。
